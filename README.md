#  AI Data Analyst App

Turn any dataset into insights instantly.

##  What it does
- Upload any CSV
- Auto generates:
  - Key Insights
  - Trends
  - Business Recommendations
- Ask questions about your data (AI-powered)

## Features
- Automated EDA
- Correlation Heatmaps
- Smart Insights Generator
- Natural Language Q&A

##  Tech Stack
- Python
- Streamlit
- Pandas
- Matplotlib / Seaborn
- Groq API (LLM)
